# AVATAR

AVATAR is een webproject waarmee je een 3D-avatar kunt maken en weergeven in de browser. Het project is opgebouwd met **HTML**, **CSS** en **[Three.js](https://threejs.org/)**. Het is ontworpen als leeroefening in 3D webtechnologie en 3D-modellering, geïnspireerd door de populaire serie *Avatar*.

## Features

- 3D avatar renderen met Three.js
- Real-time interactie (draaien, zoomen)
- Aanpasbare uiterlijk van de avatar
- Moderne en responsieve interface

## Installatie

Geen installatie van externe pakketten vereist. Je hebt alleen een moderne webbrowser nodig.

1. Download of clone deze repository:

    ```bash
    git clone https://github.com/Mahammedali2002/AVATAR.git
    cd AVATAR
    ```

2. Open `index.html` in je favoriete webbrowser.

## Gebruik

1. Open `index.html` rechtstreeks in je browser.
2. Beweeg je muis en gebruik scroll om de avatar te bekijken en te verkennen.

## Projectstructuur

- **index.html** – De startpagina
- **style.css** – Stijlen voor de interface en avatar
- **main.js** – Bevat alle Three.js logica voor 3D-weergave

## Technologieën

- **HTML5**
- **CSS3**
- **JavaScript**
- **Three.js**

## Voorbeeld

```html
<!-- index.html (fragment) -->
<div id="avatar-container"></div>
<script src="https://cdn.jsdelivr.net/npm/three@0.134.0/build/three.min.js"></script>
```
